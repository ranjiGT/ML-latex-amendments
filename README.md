Compiled PDFs are available in the [`ML-Wise2021_Nair/`](ML-Wise2021_Nair/) folder.

## Prerequisites

To compile the `.tex` sources you will need:
- A LaTeX distribution such as [TeX Live](https://www.tug.org/texlive/) or [MiKTeX](https://miktex.org/)
- A PDF viewer for reading the compiled output

Pre-compiled PDFs are already available in [`ML-Wise2021_Nair/`](ML-Wise2021_Nair/) if you do not want to build from source.
